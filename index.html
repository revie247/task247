Setup — takes 5 minutes, free
1
Go to console.cloud.google.com → Create a new project (e.g. "Task247")
2
APIs & Services → Enable APIs → search and enable Google Drive API and Google Picker API
3
APIs & Services → Credentials → Create credentials → OAuth 2.0 Client ID → Web app → Add your file URL or http://localhost to Authorised JS origins
4
Also create an API Key under Credentials → restrict it to Drive API + Picker API
5
Paste both below and click Connect
